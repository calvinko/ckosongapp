# Mini Song App

The lightweight version of the song app with support for offline.

All of the data is based off `mini/data/*`, which is generated by `scripts/mini/write_mini_song_list.py`.

Install [nvm](https://github.com/nvm-sh/nvm)

```bash
# use node v22
nvm default 22
nvm use default

# install dependencies (in mini/ folder)
npm install

# start development server
npm start

# go to localhost:3000 to view
```

**When Updating Songs**

```bash
# Go to top level folder

# install dependencies (if yarn not installed - npm install -g yarn)
yarn

# run to generate song data
yarn write-song-list
```
