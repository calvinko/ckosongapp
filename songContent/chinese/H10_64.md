# 竟有這樣的良人與專注

??怎看主）

1. 比萬有，主你更真實；
你在天上為我永活，
你和你愛環繞著我，
你更永住在我心裡。

2. 比萬有，主，你更真實；
你晝夜專注向著我（attend to me），
你向我所懷的愛念，
比海沙天星更多。

3. 你全心全意的愛我，
好像宇宙再無別的；
多寶貴你是無限的，
能無限個人性愛我。
（能無限專心愛我。）

4. 從亙古你有美麗愛夢，
你心珍藏愛的秘密，
宇宙中你最最要我，
永遠作你至相配佳偶。

5. 你滴血的心在十架打開，
你珍貴能為我死！
我今能與你永同活，
啊！無盡的愛河情愛路。

6. 你愛是耶和華的烈焰，
比彩虹更多姿采。
你恆動的情心、愛念，
永吸引、奪去我心。

19-11-2006 睡中醒時，麗港城家
