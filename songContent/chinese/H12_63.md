# 天天感恩大UV歡呼

詞：余光昭　曲：S C Foster

1, 3. 主環抱我經過水火，
進到豐富之地；
祂使所經流淚幽谷，
變為泉源之地。
獅子坑、火窰，有主親陪伴！
祂使我在夜間歌唱，
風雨中與祂擁抱！

2. 苦難越大，恩典越多，
賞賜實難計算！
為主受苦是我珍寶，
我樂為祂飲苦杯。
苦難、挑戰強，勝利更榮耀！
為主被人毀謗、敵擋，
加添我喜樂情由。
（為我永遠增榮甜！）

4. 苦難越大，歌聲越亮，
夜間歌唱、歡騰，
看見前面喜樂、榮耀，
凡事謝恩，大UV！（歡呼）
天天被恩感，明祂柔細愛；
體會主情愛多深厚，
天天感恩大UV！（天天大UV歡呼！）

18-12-2010　上大學站途中，香港
